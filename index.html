What is Markdown?

Markdown is a simple markup language that allows you to focus your time and attention on your content and its structure, without having to worry about formatting. Once you've finished creating your content, you can use a tool like Pandoc to convert it to whatever format you need (e.g., HTML, PDF, DOCX, EPUB).

Why Use Markdown?

Markdown has some real advantages:

  • It's lightweight.
  • It's plain text, so it can be written and opened on any device running any operating system.
  • Converting from Markdown text to particular file formats — e.g., DOCX, HTML, PDF — is a simple matter.

So, if you're someone who frequently writes text that needs to be published to a number of different formats, you'll likely find Markdown incredibly useful. Write your text once, then export it to whatever file formats you need.

On the other hand, if you frequently need to create documents with highly complex visual formatting, and you don't often need to output your work to multiple file formats, Markdown may not be the best tool for your purposes.

Is Markdown Difficult to Learn?

Not only is Markdown not difficult to learn, it's really quite simple. In fact, you may find you know a good bit of the syntax already. Have you ever enclosed a word in asterisks in a Facebook post because italics aren't available?

That's exactly how you create italics in Markdown. When you write *italics* in your Markdown text, it will output as italics.

Creating Lists in Markdown

Likely you already know how to make lists, too. To create an unordered list that looks like this:

  • Item 1
  • Item 2
  • Item 3

You need only write:


                * Item 1
                * Item 2
                * Item 3

Ordered lists are just as easy. To get:

  1. Item
  2. Item
  3. Item

Just write:


                1. Item
                2. Item
                3. Item

Creating Links in Markdown

Links are easy to create, too. Simply enclose the text you want to be clickable in square brackets, followed immediately by the link in parentheses, like so: 

            [clickable text](URL to load on click)

Styling Text in Markdown

We already saw one example of how to style text (styling it as italic) above. Other styling options are also pretty simple. For example, to make text bold, just enclose it in double asterisks: **This text is bold.**

Markdown understands HTML, too, so you could make your link by writing:<a href="URL to load">Clickable text</a>.

But links are so easy to create in Markdown that there's no need to use the HTML in that instance. It's included here only as an example.
